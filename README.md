# pnl-saas-boilerplate

> **Status: In progress — not ready for use**

Multi-tenant SaaS boilerplate for the ProtoNodeLabs stack. Battle-tested in production before public release.

## Planned stack

- Next.js 16 (App Router)
- Prisma + Neon (serverless Postgres)
- Organization / workspace / member model
- Auth (Clerk or NextAuth)
- Stripe billing hooks
- Resend transactional email (using [react-email-kit](https://github.com/ProtoNodeLabs/react-email-kit))
- Vercel deployment-ready

## Status

Building this out in a real project first. Will open-source once the patterns are validated.
